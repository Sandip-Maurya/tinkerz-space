# Image Requirements for Tinkerz Space Website

## Theme Overview

All images must align with the **futuristic tech aesthetic** of Tinkerz Space:

- **Visual Style**: Modern, cutting-edge, innovative, and inspiring
- **Color Palette Alignment**: 
  - Primary background: Dark blue (#0a1929)
  - Secondary background: Dark blue light (#1a2f4a)
  - Accent color: Cyan (#00d4ff, #4fc3f7)
  - Text: White with various opacity levels
- **Mood**: Energetic, forward-thinking, hands-on learning, innovation-driven
- **Environment**: High-tech maker space with advanced equipment, robotics, AI, drones, and space technology
- **Atmosphere**: Professional yet accessible, inspiring creativity and innovation

### Visual Guidelines

- Images should showcase modern technology and hands-on learning
- Include elements that suggest innovation, creativity, and technical excellence
- Maintain consistency with the dark blue and cyan color scheme
- Ensure images are crisp, clear, and professionally composed
- Avoid outdated or overly traditional educational imagery
- Focus on students actively engaging with technology
- **People/Faces**: When using any face or person in images, use Indian/Pakistani/Bangladeshi origin appearance

## Screen Size Breakpoints

- **Mobile**: < 768px width
- **Tablet**: 768px - 1023px width  
- **Laptop**: 1024px+ width

## Aspect Ratios Available

The following aspect ratios are used across different sections and screen sizes:
- **3:2** (Landscape - 1.5:1)
- **2:3** (Portrait - 0.67:1)
- **1:1** (Square)
- **16:9** (Widescreen - 1.78:1)
- **9:16** (Vertical/Story format - 0.56:1)

## File Naming Convention

**Format**: `{section-name}-{breakpoint}.{extension}`

**Breakpoints**:
- `mobile` - For mobile devices (< 768px)
- `tablet` - For tablet devices (768px - 1023px)
- `laptop` - For laptop/desktop devices (1024px+)

**Extensions**:
- Primary: `.png` (PNG format)
- Alternative: `.jpg` or `.jpeg` (JPEG for photos if preferred)

**Examples**:
- `vision-mobile.png`
- `problem-tablet.png`
- `who-we-are-laptop.png`
- `facilities-mobile.png`
- `what-students-learn-laptop.png`

## Technical Requirements

### File Formats

1. **Primary Format: PNG**
   - Preferred format for all images
   - Supports transparency when needed
   - Good quality for web display
   - Use appropriate compression to balance quality and file size

2. **Alternative Format: JPEG**
   - JPEG for photographic content (use quality: 85-90)
   - Use if file size is a concern and transparency is not needed

### Image Optimization

- **Target file sizes** (PNG format):
  - Mobile images: < 200KB
  - Tablet images: < 350KB
  - Laptop images: < 500KB
  
- **Optimization techniques**:
  - Use appropriate compression levels (balance quality vs. file size)
  - Remove metadata if not needed
  - Ensure images are properly sharpened for web display
  - Use progressive JPEG encoding for JPEG images when applicable

### Image Quality Guidelines

- **Resolution**: Provide images at 2x the display dimensions for retina/high-DPI displays (dimensions listed above already account for this)
- **Sharpness**: Images should be crisp and clear, suitable for web display
- **Color profile**: sRGB color space
- **Color correction**: Ensure colors align with the dark blue and cyan theme where possible

### Delivery Location

All images should be placed in the `/public/images/` directory with the exact file names specified above.
