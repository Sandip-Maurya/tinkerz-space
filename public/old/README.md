# Image Assets Required

Place the following images in this folder (`public/images/`) with the exact file names listed below:

## Logo & Favicon

1. **logo.svg** (or **logo.png**)
   - Main logo for the navigation bar
   - Recommended size: 200x60px or similar aspect ratio
   - Format: SVG (preferred) or PNG with transparent background

2. **favicon.svg** (place in `public/` folder, not in images/)
   - Browser tab icon
   - Recommended size: 32x32px or 64x64px
   - Format: SVG (preferred) or PNG/ICO

## Content Images

3. **vision.jpg** (or **vision.png**)
   - Vision section image
   - Recommended size: 1200x900px (4:3 aspect ratio)
   - Content: Students working on electronics/projects

4. **problem.jpg** (or **problem.png**)
   - Problem section image
   - Recommended size: 1200x900px (4:3 aspect ratio)
   - Content: Hands-on robotics project

5. **who-we-are.jpg** (or **who-we-are.png**)
   - Who We Are section image
   - Recommended size: 1920x1080px (16:9 aspect ratio)
   - Content: Maker Space environment/workspace

6. **facilities.jpg** (or **facilities.png**)
   - Facilities section image
   - Recommended size: 1200x900px (4:3 aspect ratio)
   - Content: State-of-the-art facilities overview

7. **what-students-learn.jpg** (or **what-students-learn.png**)
   - What Students Learn section image
   - Recommended size: 1200x900px (4:3 aspect ratio)
   - Content: Hands-on learning experience

## Notes

- All images should be optimized for web (aim for <500KB per image)
- JPG format is recommended for photos, PNG for images with transparency
- SVG is preferred for logos and icons
- Images will be automatically optimized by Astro during build
